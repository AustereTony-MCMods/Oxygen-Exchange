0.10.0b [2020-01-17]
+ Доработан дизайн меню.
+ Изменена команда отправки запроса на обмен. Теперь для отправки запроса требуется навести прицел на
игрока и использовать команду "/oxygenc exchange -offer".
+ Добавлен кулдаун в 1 секунду для отправки пакетов - операций в меню обмена.
+ Обновлены параметры конфигурации:
 Сервер 
 - "exchange_request_expire_time_seconds" [int] - время активности запроса на обмен в секундах

+ Добавлены привилегии:
 - "exchange:allowExchange" [id - 300, type - BOOLEAN] - возможность отправлять запросы на обмен. По умолчанию любой игрок может отправлять запросы.

+ Добавлены клиентские настройки:
 Интерфейс
 - расположение меню 
 - цвета слота
 - цвет подсветки слота
